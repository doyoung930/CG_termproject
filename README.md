# 게임 개요
- 게임제목: Shape
- 게임컨셉: 플레이어가 캐릭터를 조종하여 적을 쓰러트리며 다음 방으로 진행하며 지속적으로 살아남는 1인칭 슈팅게임

# 게임 진행방식
- 스테이지 시작시 적이 생성
- 적들을 모두 처치시 다음 방으로 이동
- 방을 이동시 문이 닫히며 새로운 몬스터 스폰
- 이를 반복하는 게임
- 방의 이동은 다음 방으로 이동하더라도 원래방으로 다시 돌아오는 것으로 클리어 후 스테이지만 변경되며 스테이지에 따라 몬스터의 숫자만 변경시킴
